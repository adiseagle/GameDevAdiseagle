Projects

Fold To Shapes

A puzzle-based project exploring folding mechanics and geometric transformations.

Main controller of the folding system responsible for setting up and running the whole gameplay loop

Since the "folds" are infinitely thin I can't use Unity's collider so I made a custom collision just for this game. There's 3 versions of it: the first version is the early working prototype on a non-parallel computation, the second version uses GPU parallelization so it's 40x faster but it failed to work on mobile, and the third version uses Unity's DOTS system which ran code in parallel on the CPU — this is what the release game is using.

The "folds" also have outlines. Using textures made them look blurry, so I created a function that generates another geometry around them to make outlines that are sharp and crisp.

Car Simulation Project

A personal project of mine exploring physics and geometry.

Realistic car's steering effects of the car during stand still shifting the car slightly.

The technical highlight is the suspension physic model that I made from the ground up, it closely mimics real life car suspension behaviour based on my theory.

One of the suspension is the Macpherson Strut type. It's calculated using vector math extensively with it only need a position and a normal plane as input. It achieves the inverse kinematics through the use of iteration and clever uses of trigonometry.

This is a Double Wishbone type. It's calculated similar to the previous type but with an extra linkage. But that's not all.

This system also supports anti rollbar and it acts as a physical bar that gets twisted on 2 ends much like the real one.
